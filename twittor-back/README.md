# twittor-back
The backend part of Twittor app.

## First Setup
```
# Set up the venv
python3 -m venv venv

# Activate the venv
source venv/bin/activate 

# Install the dependencies
pip install -r requirements.txt
```

## Start
```
# Activate venv
source venv/bin/activate 

# Start the app
python app.py
```
